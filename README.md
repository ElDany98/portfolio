# Daniel Meza — Portfolio Profesional

**Cloud Architect · Platform Engineer · AI Engineering**

Sitio web personal construido con Next.js, Tailwind CSS, Neon PostgreSQL y Cloudflare Pages.

## Stack

- **Framework:** Next.js 16 (App Router)
- **Estilos:** Tailwind CSS v4 + tokens de diseño personalizados
- **UI:** Componentes propios con paleta oscura profesional
- **DB:** Neon PostgreSQL serverless + Drizzle ORM
- **Deploy:** Cloudflare Pages + GitHub Actions

## Desarrollo local

```bash
npm install
npm run dev
```

## Despliegue

El deploy se activa automáticamente al hacer push a `main` vía GitHub Actions + Cloudflare Pages.
